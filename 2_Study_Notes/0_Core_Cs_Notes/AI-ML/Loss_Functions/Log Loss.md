# Log Loss

Log Loss is a loss function used for classification.

## Simple idea

- It checks how close the predicted probability is to the correct class
- Wrong confident predictions get punished more
- A smaller log loss means better classification

## Used by

- [Logistic Regression](../Algorithms/Logistic%20Regression.md)
- [XGBoost](../Algorithms/XGBoost.md) for classification tasks

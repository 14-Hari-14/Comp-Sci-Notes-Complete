# Hinge Loss

Hinge Loss is a loss function used mainly with support vector machines.

## Simple idea

- It checks whether the prediction is on the correct side of the margin
- Correct predictions far from the margin get less penalty
- Wrong predictions get a bigger penalty

## Used by

- [Support Vector Machines (SVM)](../Algorithms/Support%20Vector%20Machines%20(SVM).md)

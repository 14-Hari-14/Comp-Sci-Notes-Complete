# Wireshark

Wireshark is a free tool that lets you see all the data moving through your network.

## What it does

- Captures network traffic (data packets)
- Shows you what information is being sent
- Helps find network problems
- Used to see if someone is spying on your connection

## Main features

- **Live capture**: Watch data moving right now
- **Filtering**: Find specific types of traffic
- **Color coding**: Easy to spot important packets
- **Detailed analysis**: See exactly what data is being sent

## How to use it

1. Open Wireshark
2. Select your network to capture from
3. Start capturing packets
4. Use filters to find what you need
5. Analyze the packets you found

## Security use

- Detect [Packet Sniffing](Packet%20Sniffing.md) attacks
- Monitor for suspicious network activity
- Understand network behavior
- Test network security

## Important note

- **Only use on networks you own or have permission to monitor**
- Using Wireshark on other people's networks is illegal
- Related to [Packet Sniffing](Packet%20Sniffing.md) but used for legitimate purposes

A backend programs with python can be written in the following steps:
1. Import dependencies
2. Create the schema with pydantic
3. Initialize global variables required
4. Load any resources required for startup
5. Create the endpoints required 


The backend architecture that is an industry standard is the 3 layer architecture which contains
1. Routes Layer
2. Service Layer
3. Data Layer


To write any **endpoint** -> any specific addressable URL which can be used by the **API (Application Program Interface)** to receive requests from the client is called an endpoint.

A **function decorator** in Python is a design pattern that allows you to **modify or extend the behavior** of a function without changing its source code. It is a function that takes another function as an argument, adds some functionality, and returns a new function (or a modified version of the original function).

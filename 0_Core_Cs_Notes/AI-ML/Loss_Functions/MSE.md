# Mean Squared Error

Mean Squared Error (MSE) is a loss function used for regression.

## Simple idea

- It checks how far the predicted value is from the real value
- It squares the error so big mistakes count more
- A smaller MSE means better predictions

## Used by

- [Linear Regression](../Algorithms/Linear%20Regression.md)
- [XGBoost](../Algorithms/XGBoost.md) for regression tasks

# AES (Advanced Encryption Standard)

AES is a symmetric encryption algorithm that is used to quickly encrypt large amounts of data. Symmetric here means that it uses a single shared key to encrypt and decrypt the data.

**Symmetric Encryption**

- **Key Usage:** One shared secret key for encryption/decryption.
- **Speed:** Very fast and efficient.
- **Use Cases:** Bulk data encryption, database security, VPNs.
- **Examples:** AES (most popular), DES, 3DES, Blowfish.
- **Weakness:** Key distribution is risky—if the key is intercepted, security is broken.

## Activation Function

Its a mathematical function applied to the output of a neuron which decides whether it should be activated or not. This allows the neural net to map / understand complicated patterns. This process of mapping complex patterns is possible due to the neuron now understanding **non linearity**. 

Without an activation function the neural net would become **linear model** and would not be able to solve complicated tasks like 
- NLP
- Image recognition
- etc

There are multiple type of activation function 3 most common are

- **tanh**
- **relu**
- **sigmoid**
